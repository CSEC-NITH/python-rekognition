### Dependencies
* Python
  * boto3
  * cv2
  * opencv
* awscli

#### How to run?

To run analysis tool:
```
  py analysis.py
```

To run face detection tool:
```
  py detection.py
```

#### What to install and how?

1. Setup AWS account
2. Setup Dependencies (Including aws credentials)
  ```
    sudo apt-get install python awscli
    pip install boto3, cv2, opencv
  ```
